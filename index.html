ViBERTa

ViBERTa is a sentiment analysis model fine-tuned from DeBERTa-v3-base on McDonald's customer reviews. It classifies reviews into positive, neutral, or negative sentiments with high accuracy. The project includes a Streamlit app for real-time predictions and is built using PyTorch and Hugging Face Transformers. It demonstrates effective fine-tuning and deployment of transformer-based models for real-world text classification tasks.

Sentiment Analysis NLP DeBERTa Hugging Face

Savor AI

SavorAI is an AI-powered restaurant menu assistant that analyzes uploaded PDF menus to provide personalized dish recommendations and interactive dining guidance. It supports natural‑language conversations, understands context‑aware queries, and factors in dietary preferences or restrictions. Built using Streamlit and Ollama with a RAG-based retrieval pipeline, it aims to simplify meal selection based on mood, price range, and availability.

Retrieval Augmented Generation Generative AI Streamlit Chatbot

MECP2 Genomic Classification

This project classifies MECP2 gene mutations as benign or pathogenic using a hybrid machine learning and deep learning approach. It integrates genomic context, mutation characteristics, and sequence-based features through machine learning and deep learning models with SMOTE balancing and Optuna tuning. Expert-derived genomic features—like trinucleotide context, flanking sequence, mutation position, and molecular impact—drive robust variant classification.

Genomics MECP2 Classification Machine Learning Bioinformatics

EEG Analysis On DEAP Dataset

We performed emotion classification with EEG signals from the DEAP dataset. The task was to comprehend human emotions through the interpretation of brainwaves. The project involved cleaning multichannel EEG signals, extracting significant features, and training machine learning models to recognize emotional states such as valence and arousal.

Brain Computer Interface Information Extraction EEG Analysis Machine Learning
